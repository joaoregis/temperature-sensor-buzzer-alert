# Temperature sensor example
Circuito que alerta através de um buzzer quando a temperatura no ambiente ultrapassa o limite digitado pelo operador.

# Componentes usados
- Protoboard
- Sensor de temperatura
- Buzzer
- Cabos
- Resistor 1 kΩ
- Teclado 4x3 ou 4x4
